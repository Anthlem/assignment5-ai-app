# Grill Me Notes

## Original idea
Build a larger study helper with many features.

## Problems found
- Too large for the assignment
- Too risky for a beginner
- Harder to test and deploy
- More likely to fail during the demo

## Final decision
Build a small Study Notes Assistant.

## What changed because of grilling
- Removed login
- Removed database
- Removed file upload in the UI
- Kept one clear feature: ask questions about prepared notes
